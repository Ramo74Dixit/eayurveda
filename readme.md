Basic Setup of express js 
==========================

-> first create a simple package.json file 
  =====> Command --->  npm init -y 
-> Required Dependencies  
  - express
  - mongoose 
  - dotenv 
  - nodemon 

  npm i express mongoose dotenv nodemon
